# Homebrew Tap for Silimon

This is a Homebrew tap for [Silimon](https://github.com/odfalik/silimon) - an Apple Silicon performance monitor for your menu bar.

## Installation

```bash
brew tap odfalik/silimon
brew install silimon
```

## Usage

```bash
silimon
```

Click the menu bar icon to see detailed metrics.
